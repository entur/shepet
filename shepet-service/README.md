# Shepet-service
Kobling mot Statens vegvesen sitt Autosys system, for å hente informasjon om Norske kjøretøy

Se https://autosys-kjoretoy-api.atlas.vegvesen.no/api-ui/index-api.html?apiId=enkeltoppslag for beskrivelse av API-et.

Dette API-et gir oss muligheter til å hente informasjon om norske kjøretøy, bl.a.
- VIN nummer
- Drivlinje (el/hybrid/diesel/bensin)
- Vekt og bæreevne
- EURO versjon
- Antall passasjerer
